Frases HTML

Em: Se utiliza para dar significado o enfasis a una parte del texto

STRONG: Se usa para resaltar en negritas alguna parte del texto

Q: Se usa para poner comillas al texto

CODE: Se usa para poner el txto como si fuera de código de algun Lenguje de programación

ABBR: Se utiliza para poner Abreviaciones

ACRONYM: Se tuliza para poner acentos acrónimos

DFN: Se usa para poner definiciones

KBD: Significa texto que debe ser introducido por el usuario

SAMP: Se utiliza para mostrar un ejemplo de una salida de un programa

VAR: Se utiliza para mostrar el texto de una variable
